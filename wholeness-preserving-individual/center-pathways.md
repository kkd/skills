# 弱いセンターへの迂回路マップ

弱いセンターに「直接向かわず、他のセンターを経由して間接的に強化する」ための迂回路。

直接アプローチが難しいとき、迂回路から問いを始める。
複数の経路が交差するところに、本人のコアが浮かび上がる。

---

## 特性2（力強いセンター）が弱いとき

「本当は何がしたいか」「自分の核心」が見えていない。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性11（粗っぽさ）| 「自分の変なこだわり・好きなところを一つ挙げると？」 |
| 特性13（空）| 「何もしない時間に、自然と考えることは何？」 |
| 特性4（交互反復）| 「一番リズムよく過ごせた日はどんな日だった？」 |
| 特性15（不可分性）| 「自分がここにいる理由、何かを感じることはある？」 |

→ 複数の問いの答えが交差するところに「自分のコア」が浮かぶ

---

## 特性3（境界）が弱いとき

NOと言えない。自分の時間・エネルギーが守られていない。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性9（コントラスト）| 「全力でやりたいことと、そうでないことは？」 |
| 特性2（力強いセンター）| 「自分の核心が守られていると感じる瞬間は？」 |
| 特性14（シンプルさ）| 「やめたら楽になりそうなことは？」 |
| 特性13（空）| 「何もしなくていい時間があったら、どんな感覚がする？」 |

→ 「失いたくないもの」が明確になると、境界の輪郭が見えてくる

---

## 特性4（交互反復）が弱いとき

休めない、リズムがない、頑張りっぱなしか停滞のどちらか。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性5（正の空間）| 「最後にぼーっとできたのはいつ？」 |
| 特性13（空）| 「静けさの中で何かが生まれた経験はある？」 |
| 特性6（よい形）| 「自分のエネルギーの波に、どんなパターンがある？」 |
| 特性1（スケール）| 「大きなサイクル（季節・年）で見たとき、どんなリズムがある？」 |

---

## 特性5（正の空間）が弱いとき

余白がない。隙間を全て埋めようとする。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性13（空）| 「何もしていない自分でいることに、どんな感覚がある？」 |
| 特性9（コントラスト）| 「力を抜いていい場所はどこか、感じることはある？」 |
| 特性3（境界）| 「詰め込まれていくのを感じるのはどんなとき？」 |
| 特性14（シンプルさ）| 「頭が騒がしくなるのはどんなとき？」 |

---

## 特性6（よい形）が弱いとき

外部の型に自分を合わせている。「こうあるべき自分」を演じている。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性11（粗っぽさ）| 「自分の特徴やクセで、むしろ気に入っているものは？」 |
| 特性7（局所的シンメトリー）| 「どんな場面で一番自然でいられる？」 |
| 特性2（力強いセンター）| 「外からの期待を全部除いたとして、何をしたい？」 |
| 特性4（交互反復）| 「一番消耗しない働き方・過ごし方はどんな形？」 |

---

## 特性7（局所的シンメトリー）が弱いとき

場面によって全く違う自分になる。一貫した「らしさ」がない。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性11（粗っぽさ）| 「どんな場所でも変わらず出てくる自分のクセは？」 |
| 特性12（共鳴）| 「自分が大事にしていると感じることが、どの場面でも一致している？」 |
| 特性2（力強いセンター）| 「どの場面でも自分だと感じる核心はある？」 |

---

## 特性8（深い相互結合）が弱いとき

関係が表面的。仕事と私生活が分断。孤立感がある。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性15（不可分性）| 「今の生活で、深くつながっていると感じる場所・人・活動は？」 |
| 特性2（力強いセンター）| 「自分の核心が相手と共鳴したと感じた最後の経験は？」 |
| 特性12（共鳴）| 「仕事でやっていることと、大事にしていることはつながっている？」 |
| 特性11（粗っぽさ）| 「自分のどんな部分を、相手は面白いと言ってくれた？」 |

---

## 特性9（コントラスト）が弱いとき

全てが同じ優先度。緩急がない。何が大事かわからない。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性2（力強いセンター）| 「絶対に失いたくないものは何？」 |
| 特性3（境界）| 「これはやりたくないと感じるのはどんなとき？」 |
| 特性14（シンプルさ）| 「一番大事なことだけ残して、他を削るとしたら？」 |
| 特性1（スケール）| 「10年後の自分から見て、今の何が重要に見える？」 |

---

## 特性10（グラデーション）が弱いとき

全か無か思考。小さな変化を認められない。完璧主義。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性4（交互反復）| 「小さな進歩が積み重なった経験はある？」 |
| 特性11（粗っぽさ）| 「不完全なまま進んだことで、良かったことはある？」 |
| 特性13（空）| 「目標がなくても、ただそこにいてよかった瞬間は？」 |
| 特性1（スケール）| 「半年前・1年前と比べて、何が変わった？」 |

---

## 特性11（粗っぽさ）が弱いとき

「あるべき自分」を演じている。個性が抑圧されている。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性2（力強いセンター）| 「誰にも見せていない自分はいる？」 |
| 特性13（空）| 「何も期待されていない状態で、自然と何をする？」 |
| 特性7（局所的シンメトリー）| 「最も自分らしいと感じる場面はどこ？」 |
| 特性15（不可分性）| 「ありのままでいていい、と感じる場所や人はある？」 |

---

## 特性12（共鳴）が弱いとき

価値観と行動が乖離。言っていることとやっていることが違う。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性2（力強いセンター）| 「一番大事にしていること、一言で言えますか？」 |
| 特性3（境界）| 「これは自分の価値観と違うな、と感じながら続けていることは？」 |
| 特性9（コントラスト）| 「自分の行動の中で、納得感の高いことと低いことは？」 |
| 特性15（不可分性）| 「自分の行動が、誰かや何かの一部になっていると感じるのはどんなとき？」 |

---

## 特性13（空）が弱いとき

静けさを恐れる。何もしていないと不安。生産性のない時間を認められない。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性5（正の空間）| 「最後にぼーっとできたのはいつ？その感覚は？」 |
| 特性4（交互反復）| 「『動く』と『止まる』を繰り返せている感覚はある？」 |
| 特性14（シンプルさ）| 「頭の中が静かになるのは、どんな状況？」 |
| 特性11（粗っぽさ）| 「何も生産しなくてもいい、と感じる場所はある？」 |

---

## 特性14（シンプルさと静謐さ）が弱いとき

頭の中がうるさい。判断に迷い続ける。後悔が止まらない。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性2（力強いセンター）| 「自分の中心に戻ると、何が見えてくる？」 |
| 特性13（空）| 「何も考えなくていい時間に、どんな感覚になる？」 |
| 特性9（コントラスト）| 「今の状況の中で、本当に大事なのはどれか一つ？」 |
| 特性3（境界）| 「手放したら楽になるものはある？」 |

---

## 特性15（不可分性）が弱いとき

孤立感。「自分は周囲と別物」という感覚。根付いていない。

| 迂回路 | 入り口の問い |
|--------|------------|
| 特性8（深い相互結合）| 「誰かや何かに、深くつながっていると感じる瞬間はある？」 |
| 特性12（共鳴）| 「自分の行動が、誰かの何かと響き合っていると感じるのはどんなとき？」 |
| 特性11（粗っぽさ）| 「自分の個性が、誰かに喜ばれた経験は？」 |
| 特性2（力強いセンター）| 「自分の核心が、周囲と何かをつないでいる感覚はある？」 |
