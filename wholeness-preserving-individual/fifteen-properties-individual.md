# 15特性 × 個人の生

個人の生き方・日常・仕事に適用したアレグザンダーの15の生命特性。
各特性に「強い状態」「弱さの兆候」「問いの候補」を記載する。

エージェントはこのファイルから問いを選び、一度に1〜2個だけ使う。

---

## 特性1：スケールの段階性

**個人の生における意味**: 人生の大きな意味・使命から、年・季節のサイクル、週・日の習慣、瞬間の選択まで、
自然な階層がある。「なんのために生きているか」と「今日何をするか」がつながっている。

| | 内容 |
|---|---|
| **強い状態** | 大きな目的と日々の行動がつながっている。「これが今日の自分の一歩だ」と感じられる |
| **弱さの兆候** | 毎日がフラットに続く。「大事なこと」と「そうでないこと」の粒度差がない。意味の層がない |

**問いの候補**:

1. 「今年を振り返ったとき、『大きく動いた年だった』と感じる部分はありますか？それはどんなことでしたか？」
2. 「毎日やっていることの中に、『これは長い目で見て意味があると思う』ことはありますか？」
3. 「もし今から10年後、今日の自分を振り返るとしたら、どんなことに意味があったと思うでしょう？」
4. 「日々の忙しさと、本当に大事なことの間に、ギャップを感じることはありますか？」

---

## 特性2：力強いセンター

**個人の生における意味**: 「これが自分だ」という核心がある。
どんな状況でも戻れる自分の中心がある。

| | 内容 |
|---|---|
| **強い状態** | 「自分はこういう人間だ」という感覚がある。外からの要求に振り回されても、自分の核心に気づける |
| **弱さの兆候** | 「本当は何がしたいか」が答えられない。毎日が外部の要求に引き回されている。自分がわからない |

**問いの候補**:

1. 「最近、自分が自分らしいと感じた瞬間はありましたか？それはどんな状況でしたか？」
2. 「子どもの頃から変わっていない、自分の『変なところ』や『こだわり』はありますか？」
3. 「疲れているとき、どんなことをしていると少し回復する気がしますか？」
4. 「誰かに『あなたらしいね』と言われたとき、何についてそう言われましたか？」
5. 「もし明日から仕事も役割も全部なくなったとしたら、自分は何をしているだろうと思いますか？」

---

## 特性3：境界

**個人の生における意味**: 「自分がすること・しないこと」の輪郭がある。
自分の時間・エネルギー・価値観に境界線がある。

| | 内容 |
|---|---|
| **強い状態** | 大事なことを守るためにNOと言える。自分の輪郭がある |
| **弱さの兆候** | 全ての要求にYesと言う。「自分の時間」がない。他者の感情や期待を自分のものとして引き受けてしまう |

**問いの候補**:

1. 「最近、断ったことはありましたか？断れなかったことはありましたか？」
2. 「『これは自分がやることじゃないな』と感じるとき、どんな状況ですか？」
3. 「誰かの要求に応えた後、消耗感が強いのはどんなときですか？」
4. 「もし今の生活で一つだけ『やめていい』とするなら、何を選びますか？」
5. 「自分のエネルギーが守られている、と感じるのはどんな状況ですか？」

---

## 特性4：交互反復

**個人の生における意味**: 活動と回復のリズムがある。
頑張りと解放、インプットとアウトプット、集中と拡散が自然に交互に来る。

| | 内容 |
|---|---|
| **強い状態** | 「頑張る」と「回復する」のリズムが自然に循環している。一日・週・季節にリズムがある |
| **弱さの兆候** | 休まない（または頑張れない）。インプットとアウトプットのバランスが崩れている。リズムがない |

**問いの候補**:

1. 「最後に『ちゃんと休めた』と感じたのはいつでしたか？」
2. 「今の生活で、一番リズムよく過ごせているのはどんな時間ですか？」
3. 「最近、自分の中から何かが出てきた（アイデア・感情・言葉・創造）という感覚はありますか？」
4. 「一日の中で、エネルギーが上がる時間帯と下がる時間帯はありますか？それを活かせていますか？」
5. 「頑張りすぎているな、と感じるのはどんな状況ですか？」

---

## 特性5：正の空間

**個人の生における意味**: 「何もしない時間」「移行の時間」「余白」が意味を持つ。
空き時間・隙間・沈黙が、次の何かを生む。

| | 内容 |
|---|---|
| **強い状態** | 余白から新しい考えや感情が生まれる。「何もしない時間」を罪悪感なく過ごせる |
| **弱さの兆候** | 予定が詰まりすぎていて余白がない。空き時間を全て埋めようとする。ぼーっとすることができない |

**問いの候補**:

1. 「最近、何もしないでぼーっとしていた時間はありましたか？その時間はどんな感じでしたか？」
2. 「何か思わぬアイデアや気づきが浮かんだとしたら、それはどんな状況でしたか？」
3. 「もし今週、何も予定を入れない半日があったとしたら、どう使いたいですか？」
4. 「『移動中』『待ち時間』など、隙間の時間をどう過ごしていますか？」
5. 「何かをしていないと不安になることはありますか？それはどんなときですか？」

---

## 特性6：よい形

**個人の生における意味**: 自分の生き方・働き方が自分の性質・強み・リズムに適応している。
外部の「あるべき姿」ではなく、自分の「形」で生きている。

| | 内容 |
|---|---|
| **強い状態** | 仕事の仕方・生活のスタイルが自分の特性にフィットしている。「自分らしい形で生きている」感覚 |
| **弱さの兆候** | 「こうあるべき自分」という外部の型に自分を合わせている。自分に合っていないやり方を無理に続けている |

**問いの候補**:

1. 「今の働き方・生活スタイルで、『これは自分に合っている』と感じる部分はありますか？」
2. 「逆に、『なんかこれは自分に合っていないな』と感じているのはどんな部分ですか？」
3. 「他の人の働き方・生き方を見て、羨ましいと思うのはどんな部分ですか？」
4. 「『自分はこういう人間だから、こういうやり方が合っている』と感じることはありますか？」
5. 「最もパフォーマンスが高く、かつ消耗しない状態のとき、どんな環境・状況にいますか？」

---

## 特性7：局所的シンメトリー

**個人の生における意味**: 仕事・家庭・一人の時間に一貫したパターンがある。
「らしさ」が日常の各所に現れている。どこにいても自分でいられる。

| | 内容 |
|---|---|
| **強い状態** | どんな場面でも自分の「らしさ」が自然に出ている。場面が変わっても核心が一定している |
| **弱さの兆候** | 場面によって全く違う自分になる。「どれが本当の自分か」がわからない。仮面をつけている感覚 |

**問いの候補**:

1. 「仕事中の自分と、一人でいるときの自分に、大きなギャップを感じることはありますか？」
2. 「どんな場所・誰といても、変わらず出てくる自分のクセや反応はありますか？」
3. 「『あ、今仮面をつけているな』と感じる瞬間はありますか？それはどんな状況ですか？」
4. 「どんな場面でも一番自然でいられるのは、どんな状況のときですか？」

---

## 特性8：深い相互結合

**個人の生における意味**: 重要な他者との関係が深く有機的につながっている。
仕事と私生活、自分の内側と外側の世界が有機的に影響し合っている。

| | 内容 |
|---|---|
| **強い状態** | 重要な関係が深く、互いに影響し合っている。仕事と私生活が意味でつながっている |
| **弱さの兆候** | 関係が表面的。仕事と私生活が完全分断か、どちらかがもう一方を侵食している。孤立感がある |

**問いの候補**:

1. 「最近、誰かと深い話をしましたか？それはどんな話でしたか？」
2. 「仕事でやっていることと、自分が大事にしていることはつながっていますか？」
3. 「関係の中で、互いが影響し合って変化していると感じる人はいますか？」
4. 「仕事中の自分の状態が、家族・友人との関係に影響していますか？どんな影響ですか？」
5. 「今の生活で、自分が本当に大事だと思うつながりはどれですか？」

---

## 特性9：コントラスト

**個人の生における意味**: 「大事なこと」と「どうでもいいこと」のコントラストが明確。
全力の時間と手を抜く時間がある。軽重の差がある。

| | 内容 |
|---|---|
| **強い状態** | 「ここは全力」「ここは手を抜いてもいい」が意識的に選べている。緩急がある |
| **弱さの兆候** | 全てが同じ優先度。常に同じテンションで生きている。「大事なこと」が埋もれている |

**問いの候補**:

1. 「今の生活で、絶対に妥協したくないことは何ですか？」
2. 「逆に、『別にここは適当でもいい』と思っているのはどんな部分ですか？」
3. 「全力で取り組んだと思えた最後の経験は何でしたか？」
4. 「何でも同じようにこなさなければと感じることはありますか？」
5. 「今の状況で、もし何かの力を抜いていいとしたら、どこを抜きますか？」

---

## 特性10：グラデーション

**個人の生における意味**: 変化が段階的に起きている。成長に連続性がある。
「昨日より今日の自分が少し育っている」という感覚がある。

| | 内容 |
|---|---|
| **強い状態** | 小さな変化が積み重なっている。成長に連続性を感じる |
| **弱さの兆候** | 「全か無か」思考。完璧でなければ意味がない。急進か停滞の二択。小さな進歩を認められない |

**問いの候補**:

1. 「最近、去年と比べて変わったと感じることはありますか？どんな変化ですか？」
2. 「小さな一歩を積み重ねることに、心地よさを感じますか？それともフラストレーションを感じますか？」
3. 「自分の成長を実感できるのは、どんな瞬間ですか？」
4. 「完璧でないことに、耐えられないと感じることはありますか？」
5. 「今から半年後、どんな小さな変化が起きていたらいいと思いますか？」

---

## 特性11：粗っぽさ

**個人の生における意味**: 自分の個性・クセ・「変なところ」が生きている。
不完全さ・偏り・独特さが豊かさになっている。

| | 内容 |
|---|---|
| **強い状態** | 自分の「変なところ」を認め、それが強みになっていると感じる。均質化を目指していない |
| **弱さの兆候** | 「こうあるべき自分」を演じている。均質化された「良い人」「できる人」を目指している |

**問いの候補**:

1. 「自分の『変なところ』や『他の人と違うところ』で、実は気に入っているものはありますか？」
2. 「他の人から見て『あなたらしいな』と言われるのはどんなことですか？」
3. 「自分の『欠点』だと思っていたことが、実は強みになった経験はありますか？」
4. 「誰かの目を気にせず、自分のやりたいようにやっていいとしたら、何をしますか？」
5. 「自分の変なこだわり、趣味、好きなものを一つ教えてもらえますか？」

---

## 特性12：共鳴

**個人の生における意味**: 価値観・思考・行動・言葉が一致している。
「言っていることとやっていることが同じ」。自分の内と外がつながっている。

| | 内容 |
|---|---|
| **強い状態** | 大事にしていることが、行動・言葉・選択に現れている。内外の一致感がある |
| **弱さの兆候** | 場面によって全く違う価値観で動く。価値観と行動が乖離している。「言っていることと違う」を感じる |

**問いの候補**:

1. 「自分が大事にしていると思うことが、実際の行動に現れていると感じますか？」
2. 「『これは自分の価値観と違うな』と感じながらも続けていることはありますか？」
3. 「人に話す自分の話と、一人で考えている自分の話に、ズレを感じることはありますか？」
4. 「自分の内側（感じていること・大事にしていること）と外側（やっていること・言っていること）が一致している、と感じる場面はどんな時ですか？」

---

## 特性13：空

**個人の生における意味**: 「何もしない」「ただある」時間がある。
静けさ・沈黙・内省の空間から力が生まれる。

| | 内容 |
|---|---|
| **強い状態** | 「何もしない時間」を持てる。その静けさから洞察・意欲・創造が生まれる |
| **弱さの兆候** | 常に何かをしていないと不安。静けさを恐れる。「生産性のない時間」を認められない |

**問いの候補**:

1. 「最近、何も考えずに静かにしていた時間はありましたか？その感覚はどうでしたか？」
2. 「何もしていない自分でいることに、罪悪感を感じることはありますか？」
3. 「一人で、静かにいることは好きですか？苦手ですか？」
4. 「内省の時間から、何か大事なことが見えてきた経験はありますか？」
5. 「もし何もしなくていい時間が丸一日あったとしたら、どんな感覚がしますか？」

---

## 特性14：シンプルさと静謐さ

**個人の生における意味**: 内的な平和がある。
複雑な状況でも自分の中心から判断できる。頭の中が静か。

| | 内容 |
|---|---|
| **強い状態** | 心が落ち着いている状態から判断できる。「わかった」という感覚がある。内的な騒音が少ない |
| **弱さの兆候** | 常に何かが気になる。判断に迷い続ける。後悔が途切れない。頭の中がうるさい |

**問いの候補**:

1. 「最近、自分の中が静かだ、と感じた瞬間はありましたか？それはどんな状況でしたか？」
2. 「何かを決めるとき、迷いが長く続くことが多いですか？それとも比較的スッと決まることが多いですか？」
3. 「頭の中が騒がしいと感じるのは、どんなときですか？」
4. 「後悔や反省が頭をぐるぐるすることがありますか？どんなことについてですか？」
5. 「『ああ、これでいい』と感じた最後の瞬間はいつでしたか？何についてでしたか？」

---

## 特性15：不可分性

**個人の生における意味**: 自分が周囲（人・場所・仕事・自然・コミュニティ）と有機的につながっている感覚。
「自分は孤立した存在ではない」という感覚。

| | 内容 |
|---|---|
| **強い状態** | 自分が周囲の一部として存在している感覚がある。生き方が周囲の人・場所・時代と有機的につながっている |
| **弱さの兆候** | 孤立感・疎外感。「自分は周囲と別物」という感覚。生き方が周囲から浮いている。根付いていない感じ |

**問いの候補**:

1. 「今の自分が、誰かや何かの一部だと感じる瞬間はありますか？それはどんなときですか？」
2. 「自分がここにいる理由、ここに根付いている感覚がありますか？」
3. 「孤独を感じることはありますか？それはどんな状況でですか？」
4. 「今の生活の中で、自分が誰かや何かと深くつながっていると感じる関係・場所・活動はありますか？」
5. 「自分の存在が、誰かや何かに影響を与えていると感じることはありますか？」
